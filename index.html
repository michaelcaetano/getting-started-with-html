The Basics of The Internet

The World Wide Web is a vast collection of documents that are comprised of HTML or HyperText Markup Language. HTML is essentially what "glues" everything together as the main type of document on the Web.

HTML - HyperText Markup Language

HTML is made up of:

  • Text content (what you see)
  • Markup (what it looks like)
  • References to other documents (i.e images, videos, etc.)
  • Links to other pages
Introductory Lesson: Keys to Remember
  1. Computers are dumb.
  2. The Internet was invented in the early 1990's and is comprised of some 30 billion pages.
  3. Elements: everything within a set of opening and closing tags
  4. Attribute: property of HTML element
  5. If unsure about which tag to use when coding be sure to reference HTML tags to solve the dilemma.
